Book User Ratings and Data

Books are a popular source of entertainment for many, and thanks to the dataset provided from Kaggle, we can organize and clean data provided by users.

Using the datasets Books, Ratings, and Users, we start preparing queries through T-SQL to display on Tableau. We perform data cleaning of unnecesary data such as 0-star ratings, duplicate ISBNs, or missing publication by using CTEs and temporary tables.

For Users, the biggest problem is formatting the country of origin. Many user inputs have mispelled words, lowercased countries, redundant words (i.e "us of a"), and more. To best address the issue, a table from Trainmaster contians a clean format for each country. With that, first any user entries that use the same acronym or country spelling are formatted to the respective country listing. Afterwards, the Soundex function is used in order to best match the user entries to the listing of countries. Finally, remaining values are grouped to see which remaing entries are the most submitted, and they are manually matched. All remaining entries are set as null.

Now, with the data being prepped and organized, we can upload them to Excel sheets and send them over to Tableau.
